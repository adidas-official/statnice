Programovací jazyk na backendu aplikace. Umožňuje komunikaci s databázavými systémy. Přijímá http požadavky z frontendu. Vytvořeno pro generování dynamických stránek. Dnes verze 8. Nejpoužívanější jazyk pro webservery. 
Postaveno na C++. Syntaxe je mnohdy podobná, podpora OOP.
K db se přistupuje pomocí funkcí, například mysqli nebo novější PDO, které je pro všechny typy DBMS. DPO podporuje parametrizované query jako ochranu před code injections.
Nativní funkce pro sanitaci vstupních hodnot (htmlentities, mysqlirealescape,..)
