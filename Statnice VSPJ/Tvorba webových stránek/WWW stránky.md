# Základní části
- Tvorba značkovacím jazykem HTML.
- Mezi základní patří tagy:
	- html - obsahuje všechno
	- head - obsahuje title, scripty, link 
	- body
- Ostatní používané:
	- div
	- a
	- br
	- p
	- i
	- b
	- image
# Tagy
- Párové tagy (div, p, a, i, b,...) a nepárové tagy (image, link, ...)
# Validace
- Validace se už dlouho nepoužívá. Zelená fajfka, kterou si může někdo dát na web, když je napsán ve shodě s uznávaným standardem.
