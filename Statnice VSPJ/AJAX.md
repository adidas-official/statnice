Typicky se používá v kombinaci s PHP pro dynamické stránky. Ajax pošle HTTP request asynchronně (čeká, až příjde odpověď backend serveru a potom provede změny).

- **Vlastnosti:** Asynchronous JavaScript And XML. Slouží k dynamické změně obsahu stránky bez nutnosti jejího celkového znovunačtení (reloadu). Přenáší data (text, HTML, XML, JSON či JS skripty) přes objekt `XMLHttpRequest`.
- **Příklady použití:** Interaktivní kalkulátory bez potvrzování (odesílání a příjem výpočtu na pozadí) nebo funkce automatického dokončování (našeptávač) ve formulářích při zadávání slov.
